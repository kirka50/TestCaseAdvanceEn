# Запуск проекта

### Убедитесь что у вас свободны порты 3000, 5173  

1. Установите проект
```shell
git clone https://github.com/kirka50/TestCaseAdvanceEn.git
```
2. Установите зависимости.  

```shell
npm install
```

3. Запустите мок сервер 

```shell
npm run dev:server
```

4. Запустите фронт сервер

```shell
npm run dev
```
